# LED-Lamp-Tester
Evaluatie week 5: LED lamp tester

- ESP32 Microcontroller
- BH1750 (digital lux sensor)
- BMP280 (digital temperature sensor)

Pins:
- 3.3V
- GND
- GPIO21 SDA
- GPIO22 SCL
